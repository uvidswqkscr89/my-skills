# 基础登录 测试用例

> PRD来源: prd-user-auth-v1.md | 最后更新: 2026-03-17

## 正向流程
- [ ] TC-001: 验证用户使用正确的用户名和密码可以成功登录
- [ ] TC-002: 验证登录成功后跳转到用户首页
- [ ] TC-003: 验证勾选"记住我"后下次访问无需重新登录

## 异常流程
- [ ] TC-100: 验证输入错误密码时显示"用户名或密码错误"提示
- [ ] TC-101: 验证连续输错密码5次后账号被临时锁定
- [ ] TC-102: 验证账号锁定期间无法登录并显示剩余锁定时间

## 边界值
- [ ] TC-200: 验证用户名字段为空时无法提交登录表单
- [ ] TC-201: 验证密码字段为空时无法提交登录表单
- [ ] TC-202: 验证用户名超过最大长度限制时的处理行为
