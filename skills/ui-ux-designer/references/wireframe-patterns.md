# Wireframe Patterns | 线框图模式参考

常用布局模式和 ASCII 线框图模板，供 ui-ux-designer skill 使用。

---

## 1. 左右分栏布局（Left-Right Split）

适用场景：列表 + 详情、步骤 + 预览

```
┌─────────────────────────────────────────────────────────┐
│  [页面标题]                              [操作按钮区]    │
├──────────────────────┬──────────────────────────────────┤
│  左侧面板            │  右侧面板                        │
│  (宽度: ~40%)        │  (宽度: ~60%)                    │
│                      │                                  │
│  [列表项 1] ✅       │  [主内容区]                      │
│  [列表项 2] ❌       │                                  │
│  [列表项 3] ⬜       │  [操作/预览区]                   │
│                      │                                  │
└──────────────────────┴──────────────────────────────────┘
```

## 2. 列表行展开（Expandable Row）

适用场景：步骤列表、FAQ、错误详情

```
┌─────────────────────────────────────────────────────────┐
│ ✅ Step 1: [步骤描述]                                    │
├─────────────────────────────────────────────────────────┤
│ ❌ Step 2: [步骤描述]                        [操作链接] │
│ ┌───────────────────────────────────────────────────┐   │
│ │  [展开内容区 - 错误详情/截图/说明]                 │   │
│ │  背景色区分，左侧竖线强调                          │   │
│ └───────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────┤
│ ⬜ Step 3: [步骤描述]（未执行）                          │
└─────────────────────────────────────────────────────────┘
```

## 3. 卡片预览 + 播放器（Media Preview）

适用场景：截图预览、视频封面

```
┌─────────────────────────────────────────────────────────┐
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                   │  │
│  │           [图片/截图区域]                          │  │
│  │                                                   │  │
│  └───────────────────────────────────────────────────┘  │
│  [说明文字]                          [▶ 播放按钮/链接]  │
└─────────────────────────────────────────────────────────┘
```

## 4. 状态 Badge 行

适用场景：状态标签、优先级标记

```
[图标] [主文字]                    [Badge: 状态]  [操作...]
```

Badge 颜色约定：
- 成功/通过：绿色 `#52C41A` 背景
- 失败/错误：红色 `#FF4D4F` 背景
- 进行中：蓝色 `#1890FF` 背景
- 未执行/跳过：灰色 `#D9D9D9` 背景
- 警告：橙色 `#FA8C16` 背景

## 5. 空状态（Empty State）

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│                    [插图/图标]                           │
│                                                         │
│              [空状态标题文字]                            │
│           [说明文字，引导用户下一步]                     │
│                                                         │
│                  [主要操作按钮]                          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 6. 加载态（Loading State）

- 骨架屏（Skeleton）：用灰色矩形占位，模拟内容布局
- Spinner：居中旋转图标，适合小区域
- 进度条：适合有明确进度的操作

## 7. 错误/降级态（Error / Fallback State）

```
┌─────────────────────────────────────────────────────────┐
│  ⚠️  [错误标题]                                          │
│  [错误说明，告知用户发生了什么]                           │
│  [重试按钮]  [降级操作按钮]                              │
└─────────────────────────────────────────────────────────┘
```

## 交互标注约定

在线框图描述中使用以下标注：

| 标注 | 含义 |
|------|------|
| `[hover]` | 鼠标悬停触发 |
| `[click]` | 点击触发 |
| `[auto]` | 页面加载自动触发 |
| `[→ 跳转到 X]` | 导航到另一个屏幕 |
| `[展开/折叠]` | 切换显示状态 |
| `[disabled]` | 不可交互状态 |
| `⚠️ 关键节点` | 对应旅程图中的高优痛点 |
